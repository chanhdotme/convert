Chuyển Đổi Băng Thông
& Dung Lượng Mạng

Công cụ chuyển đổi toàn diện: bit, byte, Kbps, Mbps, Gbps, KB, MB, GB — cùng bảng quy đổi và ví dụ thực tế.

bit (b) — đơn vị nhỏ nhất, 0 hoặc 1
Byte (B) = 8 bit
Mbps = megabit/giây (tốc độ mạng)
MB/s = megabyte/giây (tốc độ file)

Công thức chuyển đổi
Băng thông → Tốc độ file
MB/s = Mbps ÷ 8
GB/s = Gbps ÷ 8
Vì 1 Byte = 8 bit. Khi nhà mạng nói "100 Mbps", tốc độ tải file thực tế tối đa là 12.5 MB/s.
Tốc độ file → Băng thông
Mbps = MB/s × 8
Gbps = GB/s × 8
Phần mềm tải về hiện "10 MB/s" nghĩa là đang dùng 80 Mbps băng thông.
Quy đổi đơn vị dữ liệu
1 KB = 1,000 B (SI)
1 MB = 1,000 KB
1 GB = 1,000 MB
Chuẩn thập phân (SI / nhà mạng, nhà sx ổ cứng). 1 GB = 1,000,000,000 byte.
Chuẩn nhị phân (IEC)
1 KiB = 1,024 B
1 MiB = 1,024 KiB
1 GiB = 1,024 MiB
Windows dùng chuẩn này: ổ 1 TB thực tế chỉ hiện ~931 GiB trong Explorer.

Ví dụ thực tế có lời giải
🌐
Gói Internet 100 Mbps
→ Tải file thực tế được bao nhiêu MB/s?
1
Nhà mạng cung cấp 100 Mbps (megabit/giây)
2
1 Byte = 8 bit → chia cho 8
3
100 ÷ 8 = 12.5 MB/s
✅ 100 Mbps = 12.5 MB/s tải file
📥
Phần mềm tải 25 MB/s
→ Đang dùng bao nhiêu Mbps băng thông?
1
Tốc độ tải hiện tại: 25 MB/s
2
Nhân với 8 để ra megabit
3
25 × 8 = 200 Mbps
✅ 25 MB/s = 200 Mbps băng thông
⚡
Cáp Gigabit (1 Gbps)
→ Tải file tối đa bao nhiêu GB/s?
1
Gigabit Ethernet: 1 Gbps
2
1 Gbps = 1,000 Mbps ÷ 8 = 125 MB/s
3
125 MB/s = 0.125 GB/s
✅ 1 Gbps = 125 MB/s = 0.125 GB/s
📡
WiFi 6 tốc độ 500 Mbps
→ Tải game 50 GB mất bao lâu?
1
500 Mbps ÷ 8 = 62.5 MB/s
2
50 GB = 50,000 MB
3
50,000 ÷ 62.5 = 800 giây ≈ 13.3 phút
✅ 500 Mbps tải 50 GB game ≈ 13 phút

🔑
Quy tắc ÷8
Mbps ÷ 8 = MB/s
Luôn chia 8 khi đổi bit→byte. Đây là quy tắc quan trọng nhất.
💡
100 Mbps thực tế
= 12.5 MB/s
Gói cáp quang phổ biến nhất Việt Nam. Tải file tối đa 12.5 MB mỗi giây.
⚡
1 Gbps = 1000 Mbps
= 125 MB/s
Gigabit Ethernet — tốc độ tải file thực tế tối đa 125 MB/s.
⚠️
Mbps ≠ MBps
b (thường) ≠ B (HOA)
Chữ b nhỏ = bit. Chữ B hoa = Byte. Nhà mạng dùng Mbps, phần mềm hiện MB/s.
📊
SI vs IEC
1 GB ≠ 1 GiB
1 GB (SI) = 1,000 MB. 1 GiB (IEC) = 1,024 MiB. Windows dùng GiB, nhà mạng dùng GB.
🧮
Tính thời gian tải
t = Size×8 ÷ Mbps
Ví dụ: 1 GB (1,000 MB) trên 100 Mbps = 1,000×8÷100 = 80 giây.
